<h2>What is JNANA?</h2>
</div>

**JNANA** (Sanskrit: ज्ञान) means *Deep Knowledge + Understanding.*

It is the world's first **emotionally-aware, time-layered, federated national RAG system** — engineered entirely from scratch for India's 1.4 billion citizens. It speaks their language. It understands their pain. It works even without internet.

> Every architectural decision, every pipeline stage, every UI animation, every security layer — built by hand over weeks of research and engineering. No templates. No shortcuts.

<div align="center">
<h2>The 5 Pillars No Other Platform Has</h2>
</div>

<br/>

<details>
<summary><b>Pillar 1 — Temporal Memory RAG &nbsp;·&nbsp; 2,400 years of Indian law, all indexed</b></summary>
<br/>

| Era | Period | What's Indexed |
|-----|--------|----------------|
| Ancient & Vedic | 500 BC+ | Manusmriti, Arthashastra, customary law |
| Colonial | 1857–1947 | Transfer of Property Act, CPC, IPC |
| Constitutional | 1947–2000 | Constitution of India, land reform acts |
| Digital India | 2001–2024 | RTI, PM-KISAN, Aadhaar, UPI, NEP 2020 |

</details>

<br/>

<details>
<summary><b>Pillar 2 — Federated Village RAG &nbsp;·&nbsp; 640,000 offline-first edge nodes</b></summary>
<br/>

```
National Brain  ──▶  State Nodes (28)  ──▶  District Pods (748)  ──▶  Village Edge (offline-first)
```

A farmer in a remote village with zero connectivity gets the same quality of answer as someone in Mumbai.

</details>

<br/>

<details>
<summary><b>Pillar 3 — Emotional Context RAG &nbsp;·&nbsp; Detects HOW you feel, not just WHAT you ask</b></summary>
<br/>

| Detected Emotion | Response Strategy |
|---|---|
| Distressed / Crisis | Crisis helplines appear first. Immediate action steps. |
| Angry / Cheated | Legal recourse. RTI. Consumer forum. FIR guidance. |
| Sad / Hopeless | Empathy before information. Welfare schemes. |
| Curious | Deep educational response with full legal context. |
| Optimistic | Opportunities, schemes, and success pathways. |

</details>

<br/>

<details>
<summary><b>Pillar 4 — Cross-Domain Causal RAG &nbsp;·&nbsp; Sees what departments never tell each other</b></summary>
<br/>

```
Crop failure query
  Agriculture   ──▶  PM-KISAN relief + crop insurance claims
  Finance       ──▶  Kisan Credit Card restructuring + debt protection laws
  Healthcare    ──▶  Farmer mental health support (if distress detected)
  Employment    ──▶  MGNREGA fallback income while recovering
  Legal         ──▶  Protection from illegal moneylender harassment
```

</details>

<br/>

<details>
<summary><b>Pillar 5 — Citizen-Loop RAG &nbsp;·&nbsp; The only bidirectional national knowledge system</b></summary>
<br/>

```
Citizen asks  ──▶  JNANA answers  ──▶  Citizen corrects  ──▶  Peer validation  ──▶  Knowledge updated
```

14,832 verified citizen corrections already incorporated into the knowledge base.

</details>

<h2>Phase 2 — Everything Built</h2>
<br/>
<img src="https://readme-typing-svg.demolab.com?font=Space+Mono&size=13&duration=1800&pause=400&color=138808&center=true&vCenter=true&width=600&height=30&lines=12+features+shipped+%C2%B7+0+templates+used+%C2%B7+100%25+hand-engineered" alt="stats"/>
</div>

<br/>

| Feature | Engineering Done |
|---------|-----------------|
| **Voice Input** | Web Speech API with 10 Indian language codes. Live canvas waveform visualiser. Volume detection via AudioContext analyser. Hindi, Tamil, Telugu, Bengali, Kannada + 5 more. |
| **Live India Map** | Canvas API heatmap — 30 states, population-weighted nodes, live particle bursts on every query, click-to-inspect panels, top states leaderboard. No external map library. |
| **Auth System** | Supabase Auth. PostgreSQL trigger auto-creates profile on signup. Avatar in header. Query history per user. Persistent via Zustand + localStorage. |
| **PWA Offline Mode** | vite-plugin-pwa + Workbox. Installable on Android and iOS. All assets cached. Works in villages with zero connectivity. |
| **Live Analytics Dashboard** | Recharts area, pie, bar, and 24-hour line charts. Supabase realtime. Realistic mock fallback. Live counter every 2.2 seconds. |
| **Security Layer** | Sliding window rate limiter (20 queries/hr). Input sanitiser strips 7 prompt injection patterns. API key validator. Abuse detector. Every request through `guardRequest()` — hand-written. |
| **i18n Translations** | Full UI in English, हिन्दी, தமிழ், తెలుగు, বাংলা. 60+ string keys. Language dropdown — switch and UI transforms instantly, zero reload. |
| **Supabase Migration** | One SQL file. Run once. 4 tables, RLS policies, auto-profile trigger, analytics functions, realtime, 8 seed queries. |
| **Indian Flag Logo** | Mathematically precise Ashok Chakra — 24 spokes at exact 15° intervals, inner ring, outer ring, hub, 24 tip dots. Spinning via SMIL. JN in saffron, chakra in navy, NA in green. |
| **National Insight Engine** | 8 national crisis topics. AI cross-domain reports. Root cause chains, policy gaps, breakthrough solutions. |
| **Pipeline Explorer** | Interactive 5-pillar explorer. Animated layer diagrams. Per-pillar metrics. Click-to-reveal panels. |
| **Federated Network Graph** | Canvas graph — bobbing nodes, traveling edge particles, real-time loop. No external library. |

</div>

```
jnana-app/
│
├── logo.svg                          JNANA logo — JN + Ashok Chakra + NA
├── index.html                        PWA entry + Google Fonts
├── vite.config.js                    Vite 5 + vite-plugin-pwa + Workbox
├── package.json                      React 18, Zustand, Recharts, Supabase
├── .env.example                      API key template
│
├── supabase/
│   └── migration.sql                 One file. Run once. Database ready.
│
└── src/
    ├── App.jsx                       Router + Supabase auth wrapper
    ├── index.css                     Design tokens + 12 CSS animations
    ├── constants/index.js            10 languages, 8 domains, 30 India states
    │
    ├── lib/
    │   ├── anthropic.js              Claude API — structured 7-section format
    │   ├── supabase.js               Auth + profiles + query_logs + feedback
    │   ├── ragPipeline.js            7-stage orchestrator, AI parallelised
    │   ├── security.js               Rate limiter + sanitiser + validator
    │   └── i18n.js                   5-language translations, getT() hook
    │
    ├── store/useStore.js             Zustand, persisted across refreshes
    │
    ├── hooks/
    │   ├── useAuth.js                Supabase session + auto profile
    │   ├── useEmotion.js             Emotion detection, debounced 400ms
    │   └── useVoice.js               Speech API + volume analyser
    │
    └── components/
        ├── layout/  Header · Background · UserMenu
        ├── ui/      JnanaLogo · VoiceButton · EmotionBadge · ResponseBox · StatCard
        └── pages/   Query · Map · Pipeline · Network · Insights · Dashboard · Auth
```

<h2>Tech Stack</h2>
</div>

| Layer | Technology | Decision |
|-------|-----------|----------|
| Frontend | React 18 + Vite 5 | Fastest dev experience, instant HMR |
| State | Zustand + persist | No boilerplate, survives refresh |
| Database | Supabase | Auth + PostgreSQL + Realtime in one |
| AI | Anthropic Claude Sonnet | Best empathy, citizen-facing quality |
| Charts | Recharts | Responsive SVG, composable API |
| Maps | Canvas API | Custom built, no dependency |
| Animations | CSS Keyframes + Canvas + SMIL | Full control, zero dependency |
| PWA | vite-plugin-pwa + Workbox | Offline-first, installable |
| Voice | Web Speech API | Native browser, no SDK |
| Security | Custom hand-written | Tailored rate limit + sanitise |
| Fonts | Syne + Noto Sans Devanagari | Every Indian script supported |
