# ⚙️ The 5 Revolutionary Pillars

*No other intelligence platform in the world has all five.*

</div>

<br/>

### 🟠 Pillar 1 — Temporal Memory RAG ⏳
> *2,400 years of Indian law — all indexed, all connected*

```
500 BC ──► Ancient & Vedic Law        (Manusmriti, Arthashastra)
  │
1857 ───► Colonial Legislation        (Transfer of Property Act, CPC)
  │
1947 ───► Constitutional Era          (The Constitution, IPC, Land Reform Acts)
  │
2001 ───► Digital India 2024          (RTI, PMKISAN, Aadhaar, UPI, NEP)
```
No other RAG system time-stamps its knowledge across 4 eras.
When you ask about land rights, JNANA traces the **full legal lineage** — not just today's law.

---

### 🔵 Pillar 2 — Federated Village RAG 🛰️
> *The only RAG that works without internet — built for Bharat, not just India*

```
          ┌─────────────────────────────┐
          │   National Brain (Cloud)    │ ← Full corpus, always in sync
          └──────────────┬──────────────┘
                         │
          ┌──────────────▼──────────────┐
          │  State Aggregators (28)     │ ← State-specific schemes & laws
          └──────────────┬──────────────┘
                         │
          ┌──────────────▼──────────────┐
          │  District Pods (748)        │ ← Pincode-level accuracy
          └──────────────┬──────────────┘
                         │
          ┌──────────────▼──────────────┐
          │  Village Edge (640,000)     │ ← OFFLINE-FIRST. Works with no internet.
          └─────────────────────────────┘
```

---

### 🟢 Pillar 3 — Emotional Context RAG ❤️
> *The first RAG system that detects HOW you feel — not just WHAT you ask*

| Emotion Detected | Response Strategy |
|---|---|
| 🔴 **Distressed / Crisis** | Crisis helplines FIRST. Immediate action steps. |
| 🟠 **Angry / Cheated** | Legal recourse. RTI. Consumer forum. Police. |
| 🟣 **Sad / Hopeless** | Empathy first. Welfare schemes. Community support. |
| 🔵 **Curious** | Deep educational response. Full context. |
| 🟢 **Optimistic** | Amplify opportunities. Schemes. Success stories. |

---

### 🟠 Pillar 4 — Cross-Domain Causal RAG 🔗
> *Sees the connections your government departments never talk to each other about*

```
Your question: "My crop failed."

Without JNANA:  Agriculture → PM-KISAN relief

With JNANA:
  Agriculture   → Crop insurance + PM-KISAN
       ↓
  Finance       → Kisan Credit Card + debt restructuring
       ↓
  Healthcare    → Farmer mental health hotline (distress detected)
       ↓
  Employment    → MGNREGA fallback while crop recovers
       ↓
  Legal         → Protection from illegal moneylenders
```

### 🟢 Pillar 5 — Citizen-Loop RAG 🗣️
> *The only RAG that gets smarter from citizens — bidirectional*

```
Citizen asks question

       ↓
Citizen marks: "This didn't work because..."
       ↓
Peer validation (14,832 verified corrections)
       ↓
Expert review
       ↓
Knowledge base updated
       ↓
```

# 🚀 Get It Running in 60 Seconds

</div>

```bash
# ── Step 1: Clone ────────────────────────────────────
git clone https://github.com/YOUR_USERNAME/jnana-app.git
cd jnana-app

# ── Step 2: Install ──────────────────────────────────
npm install

# ── Step 3: Configure ────────────────────────────────
cp .env.example .env
# Open .env and paste your keys:
#   VITE_ANTHROPIC_API_KEY  → console.anthropic.com → API Keys
#   VITE_SUPABASE_URL       → app.supabase.com → Settings → API
#   VITE_SUPABASE_ANON_KEY  → same page

# ── Step 4: Database (30 seconds) ───────────────────
# Go to Supabase → SQL Editor → paste supabase/migration.sql → Run

# ── Step 5: Launch ───────────────────────────────────
npm run dev
# → http://localhost:5173 ✅
```

| 🎤 | **Web Speech API** | Native browser. No SDK. No cost. |
